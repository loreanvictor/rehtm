Create [HTML](https://en.wikipedia.org/wiki/HTML) using [HTM](https://github.com/developit/htm):

```js
import { html } from 'rehtm'

document.body.appendChild(html`
  <div onclick=${() => console.log('HI!')}>
    Hellow World!
  </div>
`)
```
